Finding Local Address Information

Linux ip

ip addrshows network interfaces and information ip down IFACE; ip up IFACE resets the prescribed interface

Windows ipconfig

  • ipconfig /all prints all interface TCP/IP and physical information
  • ipconfig /release and ipconfig /renew will reset DHCP information

MacOS ifconfig

ifconfig -a prints all interface intormation
